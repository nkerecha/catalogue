ACCESS CATALOGUE

The Process

This Catalogue allows Individuals to Access, Review & Search for info on available data as well as request relevant data to be added to the catalogue.

The Data in the Catalogue is populated after review by the City of Guelph and access to it is available through a Microsoft Power BI embedded dashboard.

Dashboards on popular data have been created and requests for these can be sent In. These dashboards will allow for easy access of up-to-date information for multiple teams.

Features

Certain actions are controlled for data protection and accuracy reasons as such not available to all users.
